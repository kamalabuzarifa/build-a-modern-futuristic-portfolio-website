# Kamal Abu Zarifa - Futuristic Portfolio

A modern, futuristic, highly interactive personal portfolio built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, and Three.js.

## Features

- Premium cyberpunk-inspired UI with glassmorphism and neon highlights.
- Interactive 3D hero object (React Three Fiber).
- Floating particle background and mouse parallax glow.
- Scroll-triggered motion animations.
- Responsive navigation and layouts.
- Dark/Light mode toggle.
- Multi-language support (English/Arabic with RTL/LTR switching).
- Contact form integrated with SMTP API endpoint.
- SEO essentials (`robots.txt`, `sitemap.xml`, metadata, manifest).

## Tech Stack

### Frontend
- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber

### Backend
- Node.js + Express SMTP API (Nodemailer)

### Database
- MySQL

## Run Frontend

1. `npm install`
2. `npm run dev`
3. Open [http://localhost:5173](http://localhost:5173)
